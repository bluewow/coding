### 개념 설명

### 소수
1과 자기 자신을 제외한 다른 약수를 가지지 않는 자연수
- 예] 2,3,5,7,11

### 약수
어떤 수를 나누었을 때 나머지 없이 나늘 수 있는 수를 의미
- 예] 12의 약수는 1,2,3,4,6,12

### 최대 공약수(GCD), 최소 공배수(LCM)
- 두 수의 약수 중 가장 큰수
- 공통된 배수중 가장 작은수 (= 두 수의 곱 / 최대공약수)


### 유클리드 호제법 
- 최대공약수를 구하는 알고리즘
- A를 B로 나눈 나머지=R. gcd(A,B) = gcd(B,R) 이다
- A % B = R 
  - B % R = R1
    - R % R1 = R2 
      - R2 가 0이 될때까지 반복
